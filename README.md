# Week 10 Fixed Effects Assignment

This repository contains the full Week 10 assignment demo for:

QM 2023: Statistics II / Data Analytics

The notebook walks through pooled OLS, entity fixed effects, and two-way fixed effects using annual REIT panel data.

## Assignment Goal

By the end of this assignment, students should be able to explain:

1. Why pooled OLS can be biased in panel settings with unobserved heterogeneity.
2. What entity fixed effects remove (firm-specific average levels).
3. What time fixed effects remove (year-specific common shocks).
4. What variation remains after demeaning and why that variation identifies coefficients.

## Repository Contents

- `fixed_effects_demo.ipynb`: Main assignment notebook.
- `data/REIT_sample_annual_2004_2024.csv`: Input panel dataset.
- `requirements.txt`: Required Python packages.
- `fe_demo_*.png`: Exported figures generated by the notebook.
- `fixed_effects_visual_plan.md`: Design rationale for the new FE visuals.
- `AI_AUDIT_APPENDIX.md`: Documentation of AI tool usage, scope, and verification.

## Expected Learning Statements

Students should be able to say, in plain language:

1. Entity FE compares each firm to itself over time by subtracting its own mean.
2. Time FE removes shocks shared by all firms in the same year.
3. Two-way FE uses the leftover variation after both firm and year components are removed.

## Troubleshooting

- If a package import fails, reinstall dependencies with `pip install -r requirements.txt`.
- If charts do not render, rerun the plotting cell and verify write permissions in the repo folder.
- If a model cell fails, rerun preprocessing cells first so required columns exist.

## Instructor Note

The notebook is intentionally designed to mix real-data credibility with one synthetic decomposition visual, so students can see both practical modeling and the exact FE mechanics.
