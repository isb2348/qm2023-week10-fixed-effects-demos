# Week 10 Fixed Effects Demos

Teaching demos for Week 10 of `QM 2023: Statistics II / Data Analytics`.

## Contents

- `fixed_effects_demo.ipynb` - notebook walkthrough for pooled OLS, entity fixed effects, and two-way fixed effects
- `requirements.txt` - Python packages needed to run the notebook
- `fe_demo_*.png` - exported figures used in the demo

## Quick Start

1. Install the python and jupyter notebook vscode integration. Do not create a Python virtual environment.
2. Install packages with `pip install -r requirements.txt`.
3. Open `fixed_effects_demo.ipynb` in Jupyter or VS Code and run the cells in order.

## Topic Focus

This demo helps students compare pooled OLS to fixed effects models and interpret within-entity variation in panel data.
